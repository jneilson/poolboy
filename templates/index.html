Poolboy

A simple ChatGPT-genrated Python and Flask script that takes a single Blackboard Ultra question pool (bank) and performs the following fixes:

  • Replaces any instances of "Multiple Answer" with "Multiple Choice"
  • Within the <resprocessing><respcondition title="correct"> section of each question, removes any answer wrapped in a <not> tag
  • Removes any remaining <and> tags

Select a zip file and then specify the name of the output file. The script will create a new zip file with the name you've specified. You can then upload the question pool to an LMS and it wont be broken.
